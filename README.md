# tetris-react

## How to dev

### Setup

```bash
# ... Setup node and yarn
$ git clone git@github.com:kobakazu0429/tetris-react
$ cd tetris-react
$ yarn install
```

### Commands

```bash
# written in package.json
$ yarn dev   # Start webpack-dev-server
$ yarn build # Build to docs/
$ yarn lint  # Check type and syntax
```
